Rules

An RGB value will be given on the top of the website. After that, select the color you think is being represented.

If you get the answer, you will get 5 points. If you are wrong, you will lose one point.

If you get the answer wrong 3 times, you lose.

This is a game for web developers and designers, so they can understand, learn and practice the RGB color model.
